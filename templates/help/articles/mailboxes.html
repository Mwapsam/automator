Once a domain is verified, you can create real mailboxes (inboxes people can log into and receive mail) and aliases (addresses that forward to another inbox). Everything lives on the Mailboxes page.

You need at least one verified domain first. If the page tells you to add a domain, start with Add & configure a sending domain.

Create a mailbox

  1. Open Mailboxes.
  2. Under Create a mailbox, fill in:
    • Email — the full address, e.g. jane@yourdomain.com (the domain must be one of your verified ones).
    • Display name — optional, shown on outgoing mail.
    • Password — the mailbox login password.
    • Quota (MB) — how much storage the mailbox gets.
  3. Click Create mailbox. It provisions in the background and appears in the list.

Storage quotas & plan limits

Your plan sets a maximum number of mailboxes and a per-mailbox storage cap. If you request more storage than your plan allows, we'll automatically cap it and tell you. To raise these limits, upgrade on the Billing page.

Manage an existing mailbox

  • Change password — set a new login password for the mailbox.
  • Update quota — increase or decrease storage (within your plan's cap).
  • Delete — permanently removes the mailbox and its stored mail.

Create an alias

An alias forwards mail from one address to another — great for sales@, support@ or info@ addresses that should land in a shared inbox.

  1. Under Create an alias, enter the Alias address (e.g. sales@yourdomain.com) and the address it Forwards to (e.g. team@yourdomain.com).
  2. Click Create alias.
An alias doesn't store mail or have its own login — it simply redirects. If you need a real inbox, create a mailbox instead.
