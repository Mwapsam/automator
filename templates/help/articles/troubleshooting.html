Most setup snags come down to DNS or plan limits. Here are the issues we see most often and how to fix them.

My domain won't verify

This almost always means the DNS records aren't visible yet. Check:

  • Give it time. DNS can take minutes to a few hours to propagate. The card re-checks automatically, or click Check DNS to re-run it now. Each record shows Found / Not found yet so you can see what's missing.
  • Record type is TXT. All four records (verification, DKIM, SPF, DMARC) are TXT records.
  • Name and Value match exactly. Use the copy buttons on the domain card — a stray space or missing character will fail.
  • Host name format. Some providers append your domain automatically; if so, enter only the host part. See DNS setup.
  • Right domain. Make sure you added the records to the same domain you provisioned.

My emails aren't sending (API)

  • Domain verified? You can only send from a verified domain. Check the Domains page.
  • API key active? Generate or regenerate a key on the Domains page and send it in the X-Api-Key header.
  • Plan includes the API. The email API & SMTP relay is a plan feature. If yours doesn't include it, upgrade on Billing.
  • Monthly limit reached? Check your usage on the dashboard.
  • From address. The from must be on a verified domain for your account.

A message shows as "Failed"

Open Insights (on supported plans) to see delivery logs and the reason. Common causes are an invalid recipient address or the receiving server rejecting the mail. Fix the address and resend.

My mail lands in spam

  • Make sure SPF, DKIM and DMARC are all added and the domain is verified — these are what build sender reputation.
  • Avoid spam-trigger content and send to people who expect your mail.
  • New domains build reputation over time; start with low volume.

I can't create a mailbox

  • The domain must be verified first.
  • You may have hit your plan's mailbox limit — upgrade on Billing.
  • Requested storage above your plan's cap is automatically reduced.

A teammate didn't get their invite

  • Ask them to check spam/junk.
  • Confirm the email address on Settings → Team under "Pending invitations".
  • Invites expire after 7 days — revoke and re-send if needed.

Still stuck?

If none of the above helps, reach out — see the contact option at the bottom of any help page.
